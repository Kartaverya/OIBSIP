# OIBSIP - Java Development Internship

This repository contains the tasks completed as part of the **Oasis Infobyte Online Internship Program (OIBSIP)** under the **Java Development Domain**.

---

## Tasks Completed

### Task 1: ATM Interface
**Objective:**  
To simulate basic ATM operations using Java.

**Features:**
- User authentication (basic)
- Balance checking
- Deposit functionality
- Withdrawal functionality

**Tech Used:**  
Java, CLI

**Outcome:**  
A working ATM simulation that performs core banking operations through the command line.

---

### Task 2: Number Guessing Game
**Objective:**  
To create an interactive number guessing game.

**Features:**
- Random number generation
- User input handling
- Hints (Too High / Too Low)
- Loop-based gameplay

**Tech Used:**  
Java, CLI

**Outcome:**  
A functional game that demonstrates control flow, loops, and user interaction.
